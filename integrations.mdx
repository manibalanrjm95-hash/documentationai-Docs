## Git Repository Sync

Connect your GitHub or GitLab repositories to keep documentation in sync with code changes.

<Tabs>
  <Tab title="GitHub" icon="github">
    Grant repository access via OAuth.

    <Steps>
      <Step title="Connect Account" icon="link">
        Navigate to Integrations > GitHub and authorize Mani Balan.
      </Step>
      <Step title="Select Repo" icon="git-branch">
        Choose the repository to sync.
      </Step>
      <Step title="Configure Branch">
        Set the default branch, typically `main`.
      </Step>
    </Steps>
  </Tab>
  <Tab title="GitLab" icon="git-fork">
    Use a personal access token for authentication.

    <Steps>
      <Step title="Generate Token" icon="key">
        In GitLab, create a token with `read_repository` scope.
      </Step>
      <Step title="Add to Mani Balan">
        Paste the token in Integrations > GitLab.
      </Step>
      <Step title="Sync Repo">
        Select your project and enable auto-sync.
      </Step>
    </Steps>
  </Tab>
</Tabs>

<Callout kind="tip">
  Sync runs every 5 minutes or on webhook triggers for near real-time updates.
</Callout>

Configure in Settings > Authentication > OAuth.

## Webhooks Setup

Set up webhooks to receive events like document updates or user actions.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Go to Integrations > Webhooks > New Webhook.
  </Step>
  <Step title="Configure Payload">
    Select events: `doc.updated`, `user.created`.
  </Step>
  <Step title="Verify Endpoint">
    Mani Balan sends a challenge; respond with the payload.
  </Step>
</Steps>

Handle incoming webhooks securely.

<CodeGroup tabs="Node.js,Python">
  ```javascript
  const express = require('express');
  const crypto = require('crypto');
  const app = express();

  app.post('/webhook', express.raw({type: 'application/json'}), (req, res) => {
    const signature = `sha256=${crypto.createHmac('sha256', 'YOUR_WEBHOOK_SECRET')
      .update(JSON.stringify(req.body)).digest('hex')}`;
    // Verify signature from Headers['X-Signature']
    console.log('Event:', req.body.event);
    res.status(200).send('OK');
  });
  ```
  ```python
  import hmac
  import hashlib
  from flask import Flask, request, abort

  app = Flask(__name__)
  WEBHOOK_SECRET = 'YOUR_WEBHOOK_SECRET'

  @app.route('/webhook', methods=['POST'])
  def webhook():
      signature = hmac.new(
          WEBHOOK_SECRET.encode(),
          request.data,
          hashlib.sha256
      ).hexdigest()
      # Verify against request.headers['X-Signature']
      print('Event:', request.json['event'])
      return 'OK', 200
  ```
</CodeGroup>

## API Overview

Access integrations programmatically via REST API at `https://api.example.com/v1`.

<Request tabs="cURL,JavaScript">
  ```bash
  curl -X POST https://api.example.com/v1/integrations/github/sync \
       -H "Authorization: Bearer YOUR_API_KEY" \
       -H "Content-Type: application/json" \
       -d '{"repo": "user/repo", "branch": "main"}'
  ```
  ```javascript
  const response = await fetch('https://api.example.com/v1/integrations/github/sync', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      repo: 'user/repo',
      branch: 'main'
    })
  });
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "status": "success",
    "sync_id": "sync-abc123",
    "completed_at": "2024-01-15T10:30:00Z"
  }
  ```
</Response>

<ParamField path="repo" param-type="string" required="true">
  Full repository name, e.g., `owner/repo-name`.
</ParamField>

<ParamField query="branch" param-type="string" required="false">
  Branch to sync (default: `main`).
</ParamField>

<Expandable title="Advanced API Usage">
  Use query params for filtering and pagination on webhook lists.
</Expandable>

<Callout kind="info">
  All API requests require authentication with `YOUR_API_KEY`. Generate keys in Settings > API.
</Callout>
