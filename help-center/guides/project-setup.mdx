---

## Quick Guide

<Steps>
  <Step title="Start new project">
    ````
    Click "New Project" on the mani balan dashboard.
    Enter project name and optional description.
    ````
  </Step>
  <Step title="Add documentation pages">
    ````
    Use the "Add Page" button to create docs, or import existing files.
    ````
  </Step>
  <Step title="Invite team members">
    ````
    Go to "Collaborators" and invite others by email address.
    ````
  </Step>
</Steps>
